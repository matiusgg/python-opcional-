Ahora donde dice CODE, podemos escoger varios formatos para escribir dentro de python 3 en jupyter, cuando tengamos una primera linea
podemos colocar SHIFT + INTRO para bajar y seguir escribiendo si quierendo con otra fuente. Y ahora con CODE, podemos escribir codigo python

Entonces nos ejecutará esa linea de código.
Si hay algún error de sintaxis, nos lo dira.
